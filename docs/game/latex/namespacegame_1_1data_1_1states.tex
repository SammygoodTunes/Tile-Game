\doxysubsection{Detailed Description}
\begin{DoxyVerb}States package.

States are values that describe the condition that something is in.
A server can be starting, a player can be moving left, a connection can time out. Those are all states.
They don't define behaviour but report behaviour.
\end{DoxyVerb}
